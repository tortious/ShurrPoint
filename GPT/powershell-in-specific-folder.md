# How to Open PowerShell Window in a Specific Folder

**June 26, 2020** by Salaudeen Rajack

## Requirement
Open PowerShell window in a specific folder.

## How to Open PowerShell Window in a Specific Folder?
There are multiple ways to open a PowerShell window in a specific folder. Here are a few methods:

### Method 1: Using File Explorer
1. Open File Explorer and navigate to the folder where you want to open the PowerShell window.
2. Click on the "File" menu in the top-left corner.
3. Select "Open Windows PowerShell" and then choose "Open Windows PowerShell" or "Open Windows PowerShell as administrator".

### Method 2: Using the Context Menu
1. Navigate to the folder where you want to open the PowerShell window.
2. Hold down the Shift key and right-click on the folder.
3. Select "Open PowerShell window here" from the context menu.

### Method 3: Using the Address Bar
1. Open File Explorer and navigate to the folder where you want to open the PowerShell window.
2. Click on the address bar to highlight the path.
3. Type `powershell` and press Enter.

### Method 4: Using a Shortcut
1. Right-click on your desktop or in a folder and select "New" > "Shortcut".
2. In the "Type the location of the item" field, enter `powershell.exe -NoExit -Command "Set-Location -Path 'C:\Your\Folder\Path'"`.
3. Click "Next" and give your shortcut a name.
4. Click "Finish" to create the shortcut.

## Conclusion
Opening a PowerShell window in a specific folder can be done using various methods, including File Explorer, the context menu, the address bar, and shortcuts. Choose the method that best suits your workflow.
